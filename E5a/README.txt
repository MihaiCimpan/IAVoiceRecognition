Functionalitate:
    Scriptul din fisierul separare.py imparte setul de date in set de antrenare(SA) și set de test(ST) după un procent stabilit pentru 
    fiecare emotie in parte prin crearea a unui folder SA pentru datele de antrenare si a unui folder ST pentru datele de test.
    
La apelare sunt nevoie de 3 parametri: 
    -path(path-ul folder-urului unde se afla fisierele)
    -procentul pentru datele de antrenare
    -procentul pentru datele de test
